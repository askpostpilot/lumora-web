API Policy

Last updated: March 2026

1. Status
The SolyntraAI API is currently in preview. Endpoints, rate limits, and
features may change without notice. Do not build production systems
that depend solely on this API without contacting us first.

2. Access
API access requires an active paid plan and a valid API key generated
from your dashboard.

3. Rate Limits
- Starter: 100 requests/day
- Pro: 500 requests/day
- Agency: 2000 requests/day

4. Acceptable Use
You may not: resell API access, use it to scrape data, exceed rate limits,
or use it for any illegal purpose.

5. No SLA
We do not provide an uptime guarantee for the API at this stage.
