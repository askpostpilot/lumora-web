API Documentation (Preview)

Integrate Lumora's capabilities into your own applications. This preview outlines the endpoints and data available to you. Full API access will require an active subscription and API key.

Endpoints

  • GET /posts – Retrieve your scheduled and published posts.
  • POST /posts – Create a new scheduled post.
  • GET /analytics – Access engagement statistics.
  • GET /settings – Retrieve your account settings.

To request API access or see more documentation, please contact our team.
